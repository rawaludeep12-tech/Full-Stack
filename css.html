CSS Styling

CSS transforms a structured HTML document into a visually engaging user experience. Its power lies not just in controlling fonts and colors, but in its sophisticated layout capabilities like Flexbox and CSS Grid to build complex, responsive designs.

Live Demo: CSS Grid

A powerful 2-dimensional layout system. Here, a common "holy grail" layout is created with just a few lines of code.

Header
Sidebar
Main Content
Footer
.grid-container {
    display: grid;
    grid-template-columns: 1fr 3fr;
    grid-template-areas: "header header" "sidebar main" "footer footer";
    gap: 10px;
}

A 1-dimensional layout system ideal for distributing space among items in a container seamlessly.

    display: flex;
    justify-content: space-around;
    align-items: center;

Transitions & Animations

CSS allows for smooth state changes (transitions) and complex, keyframed movements (animations).

Transition on Hover

Hover over the box below.

Hover Me!
/* Transition Code */
.transition-box {
    transition: all 0.4s ease-in-out;
}
.transition-box:hover {
    transform: scale(1.1) rotate(5deg);
}

Keyframe Animation

A continuously looping pulse effect.

/* Animation Code */
.animation-box {
    animation: pulse 2s infinite;
}
@keyframes pulse {
    0% { transform: scale(0.9); }
    50% { transform: scale(1); }
    100% { transform: scale(0.9); }
}
