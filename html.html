Demonstration of Form Validation

HTML5 allows for client-side validation using attributes like required, minlength, pattern, and specific input types. Try submitting this form with invalid data.

Here is the code used for the validation form above:

<form>
  <!-- Required, min 5 characters -->
  <label for="username">Username:</label>
  <input type="text" id="username" required minlength="5">
  
  <!-- Required, must be an email -->
  <label for="useremail">Email:</label>
  <input type="email" id="useremail" required>
  
  <!-- Required, matches a specific pattern -->
  <label for="postcode">Postcode:</label>
  <input type="text" id="postcode" required pattern="[A-Za-z]{1,2}[0-9Rr][0-9A-Za-z]? [0-9][A-Za-z]{2}">
  
  <button type="submit">Submit</button>
</form>
